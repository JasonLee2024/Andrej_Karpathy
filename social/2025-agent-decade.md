# "10 years to solve Agents"

> Karpathy 在 X/Twitter 上关于 AI Agent 的核心论断。

## 观点

Karpathy 认为真正解决 AI Agent 问题需要 **10 年**。Agent 不仅仅是 LLM + function calling，而是需要：

- 全新的架构设计
- 可靠的长期规划能力
- 与环境的深度交互
- 安全与可控性保证
