# micrograd 社区语言移植

> 社区用 Rust、Go、JavaScript、C++、Julia 等语言移植了 [[micrograd]]。

## 代表性移植

| 语言 | 仓库 |
|------|------|
| Rust | micrograd-rs |
| Go | micrograd-go |
| JavaScript | micrograd.js |
| C++ | micrograd-cpp |
| Julia | MicroGrad.jl |

## 意义

micrograd 只有 ~100 行代码，是"理解 autograd 的最佳教材"，各语言移植进一步扩大了它的教学影响力。
